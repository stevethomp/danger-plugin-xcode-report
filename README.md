> Add your Xcode test results, warnings and errors to Danger.

## Usage

Install:

```sh
yarn add danger-plugin-xcode-report --dev
```

At a glance:

```js
// dangerfile.js
import xcodeReport from 'danger-plugin-xcode-report'

xcodeReport()
```
